# Contributing

Thank you for your interest in contributing!

This project is currently maintained primarily by a single developer, but contributions, suggestions, and improvements are welcome.

---

## 🛠️ How to Contribute

### 1. Fork the repository
Create your own fork and work on a feature branch:
```
git checkout -b feature/your-feature-name
```

### 2. Make your changes
- Keep changes focused and minimal
- Follow the existing code style
- Prefer clear, readable code over clever solutions

### 3. Test your changes
- Make sure the app builds and runs correctly
- Ensure no existing functionality is broken

### 4. Submit a Pull Request
- Provide a clear description of what you changed and why
- Reference related issues if applicable

---

## 💡 Guidelines

- Keep PRs small and focused
- Avoid introducing unnecessary dependencies
- For UI changes, include screenshots
- For major changes, open an issue first to discuss

---

## 🔥 Tech Stack Notes

This project uses:
- Firebase (Firestore, Authentication, Hosting)

If your contribution touches:
- **Security rules** → explain changes clearly
- **Data model** → describe migration or impact

---

## ❓ Questions

Feel free to open an issue for any questions or suggestions.
